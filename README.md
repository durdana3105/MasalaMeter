## ğŸ¤– AI/ML Models

### Food Recognition Model

**Architecture**: MobileNetV2 (optimized for mobile)
**Training Data**: 50,000+ images of Indian foods
**Accuracy**: 92% on test set
**Classes**: 500+ Indian dishes

```python
# Model Architecture
model = Sequential([
    MobileNetV2(weights='imagenet', include_top=False),
    GlobalAveragePooling2D(),
    Dense(1024, activation='relu'),
    Dropout(0.5),
    Dense(512, activation='relu'),
    Dense(num_classes, activation='softmax')
])
```

### Video Analysis Pipeline
1. Extract frames (2 FPS)
2. Detect food items in each frame
3. Track objects across frames
4. Aggregate predictions
5. Return unique items with confidence

### QR/Barcode Scanner
- **Library**: ZXing (Android), AVFoundation (iOS)
- **Database**: 100,000+ product barcodes
- **Fallback**: Online API lookup (OpenFoodFacts)
